JSON-CL

This tool allows you to have one changelog in JSON and format it to your needs.

Currently it supports JSON to markdown, but we will add HTML soon too.

We are using the code editor from this guide. Thanks!

This website is hosted with GitHub Pages. You can find the source code here.


The JSON input needs to be formatted correctly in order to work.

You can find alerts under the code editor, which can help you if you are stuck.
